História

Um breve resumo do que foi aprendido sobre o html nesses módulos foi que é uma linguagem

padrão para criar e estruturar páginas da web, ultilizando em conjunto com o CSS,

fazendo a base de todas as páginas da internet.

Conceito

Tags: HTML é baseado em tags. As tags são elementos usados para estruturar e formatar o conteúdo em uma página da web. Elas geralmente são escritas entre colchetes angulares ("<" e ">").

Estrutura básica: página HTML começa com a declaração DOCTYPE html que define a versão do HTML sendo usada, seguida pela tag html que envolve todo o conteúdo da página. Dentro da tag html, você terá as tags head e body.

Head: A seção head contém informações sobre o documento, como o título da página (title), metadados (meta), links para estilos (link), scripts (script), etc.

Body: A seção bodya contém o conteúdo visível da página, como texto, imagens, vídeos, links, etc.

Tags de texto: Tags como h1 a h6 para títulos de diferentes níveis, p para parágrafos, strong e em para enfatizar texto, ul e ol para listas não ordenadas e ordenadas, respectivamente.

Funcionamento do HTML

HTML, ou Hypertext Markup Language, é a linguagem essencial para criação de páginas web. Funciona através da marcação de elementos com tags,

que incluem uma abertura, conteúdo e, opcionalmente, uma tag de fechamento.

A estrutura básica de um documento HTML envolve as seções html, head, title, e body.

Tags como p para parágrafos, a para links e img dpara imagens são utilizadas para definir diferentes tipos de conteúdo.

Atributos podem ser adicionados às tags para fornecer informações adicionais.

O código HTML é interpretado pelos navegadores, que renderizam a página de acordo

com as instruções fornecidas, permitindo a visualização do conteúdo na web.
